# 🏥 Doctor Appointment System

A full-stack Doctor Appointment Management System built using the MERN stack. The application enables patients to register, browse doctors, book appointments, and manage their profiles, while doctors and administrators can efficiently manage appointments and users.

---

##  Features

### 👤 Patient
- User Registration & Login
- JWT Authentication
- Browse Available Doctors
- Book Appointments
- View Appointment History
- Update Profile
- Responsive User Interface

### 👨‍⚕️ Doctor
- Doctor Login
- View Scheduled Appointments
- Approve/Reject Appointments
- Manage Availability

###  Admin
- Admin Dashboard
- Manage Doctors
- Manage Patients
- View All Appointments
- Dashboard Statistics
